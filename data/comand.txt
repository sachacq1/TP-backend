*************************
How to use:
node index.js help -> information about commands.
node index.js list -> list all Users. 
node index.js search "User_id" -> search User by id. 
node index.js add "Nombre" "Apellido" "Email" "Password" -> create new User.
node index.js update "User_id" "Nombre" "Apellido" "Email" "Password" -> update User data.
node index.js delete "User_id" -> delete User.
*************************
